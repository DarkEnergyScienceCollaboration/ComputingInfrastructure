  Sections
  ========

  and

  Subsections
  -----------

  and

  Subsubsections
  ^^^^^^^^^^^^^^

  To add images, add the image file to the same directory containing your note
  The reST syntax for adding the image is

  .. figure:: /filename.ext
     :name: fig-label
     :target: http://target.link/url
